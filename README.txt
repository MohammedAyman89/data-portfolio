## ✅ Features

### 1️⃣ Web Scraping
- Scrapy projects for real websites
- Handles both GET and POST requests
- Uses custom headers, cookies, and request bodies
- Finds API endpoints via browser DevTools
- Simulates user behavior
- Extracts and structures data for analysis

---

### 2️⃣ Data Cleaning
- Standardizes date formats
- Removes currency symbols
- Handles missing / invalid entries
- Ensures valid quantities and prices
- Recalculates total sales
- Saves clean, ready-to-use CSVs

---

### 3️⃣ Data Transformation
- Merges multiple CSV files into single datasets
- Filters high-value sales transactions
- Adds computed discount columns
- Produces analysis-ready summary tables

---

### 4️⃣ Exploratory Data Analysis (EDA)
- Descriptive statistics
- Value counts and distributions
- Outlier detection with boxplots
- Price vs Quantity scatter plots (with hue coloring)
- Summaries by customer and product
- All visualizations saved as PNG
- Professional PDF report with all plots included
