# 📊 Retail Sales Data Analysis – EDA Project

Welcome! This project showcases **end-to-end data cleaning, transformation, and exploratory data analysis (EDA)** for retail sales data using Python.  

It’s designed as a **portfolio-quality freelance deliverable**: clean code, organized outputs, and a professional PDF report.  

---

## 📌 Project Goal

✅ Turn messy, raw sales CSV files into clean, usable datasets  
✅ Summarize key business metrics (per customer, per product)  
✅ Detect outliers and anomalies  
✅ Create clear visualizations for reporting

## ✅ Features & Deliverables

### 1️⃣ Data Cleaning
- Standardizes inconsistent date formats
- Removes currency symbols
- Handles missing/invalid data
- Ensures quantities and prices are valid
- Recalculates total sales amounts

✅ Result: clean CSV files ready for analysis

---

### 2️⃣ Data Transformation
- Merges multiple sales files
- Filters high-value sales
- Applies discount rules
- Summarizes sales by product and customer

✅ Result: transformed data for reporting

---

### 3️⃣ Exploratory Data Analysis (EDA)
- Descriptive statistics
- Price and quantity distributions
- Outlier detection with boxplots
- Grouped summaries by customer and product
- Scatter plots showing price vs. quantity patterns

✅ Result: ready-to-use insights

---

### 4️⃣ Visualization & Reporting
- All plots saved as **individual PNG images**
- Combined **professional PDF report** with all visualizations
- Summaries saved as **CSV** for easy analysis in Excel or BI tools

✅ Result: client-friendly deliverables
